# Movie App

Full-stack movie application with React frontend and ASP.NET Core backend.

## Features
- Browse movies
- Search functionality
- Add/Edit movies
- Image uploads
- Admin Panel Configuration

## Tech Stack
- Frontend: React, CSS
- Backend: ASP.NET Core, Entity Framework Core
- Database: SQL Server

## Deployment
- Frontend: GitHub Pages/Vercel/Render
- Backend: Heroku/Railway/Render
